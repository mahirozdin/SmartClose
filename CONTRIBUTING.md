# Contributing to SmartClose

Thanks for your interest in contributing.

## Development
1. Open `SmartClose.xcodeproj` in Xcode.
2. Select the **SmartClose** scheme.
3. Build and run.

## Principles
- Keep changes conservative and safe.
- Avoid breaking macOS conventions unless explicitly intended.
- Prefer native APIs and minimal dependencies.

## Pull Requests
- Explain the user impact.
- Include tests where possible.
- Update docs if behavior changes.
